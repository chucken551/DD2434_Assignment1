Packages required sklearn, numpy, scipy, matplotlib for this code. 
A class called DataGenerator() gets the Zoo data and separate the unimportant parts.
With the data object you can fetch the data, a 101x16 matrix and you can fetch the
corresponding type and animal lists. There are several functions for each transformation(PCA, MDS and ISOMAP).
One function to plot the graphs.

Run the code to plot three graphs. For ISOMAP the K nearest neighbors can be change in the
top of main().
